Microsoft представила DirectSR для объединения технологий супер-разрешения.

Microsoft объявила о выпуске предварительной версии DirectSR — нового API, созданного для стандартизации поддержки технологий супер-разрешения (SR) в заголовках Direct3D 12. Компания сотрудничала с AMD, Intel и NVIDIA для разработки этого API, позволяющего разработчикам без проблем использовать их технологии масштабирования через общий набор входных и выходных данных. DirectSR нацелен на упрощение интеграции технологий Super Resolution (SR), предоставляя единый код для использования DLSS Super Resolution, FidelityFX Super Resolution и XeSS. Путем предоставления этих технологий через один интерфейс интеграция SR становится более простой и масштабируемой, позволяя разработчикам реализовывать SR один раз и использовать его на широком спектре устройств под управлением Windows.
